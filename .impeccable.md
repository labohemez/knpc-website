## Design Context

### Users
- **주요 사용자**: 강남교회 교인 (전 연령대, 특히 고령 사용자 비율 높음)
- **부차 사용자**: 교회를 방문하려는 새신자, 외부 방문자
- **사용 맥락**: 예배 시간 확인, 설교 영상 시청, 공지사항 확인, 교회 정보 탐색
- **핵심 Job**: 교회 생활에 필요한 정보를 빠르고 편하게 찾는 것

### Brand Personality
- **3단어**: 따뜻한 · 신뢰 · 전통
- **톤**: 안정감 있고 격식을 갖추되 차갑지 않은 따뜻한 어조
- **감정 목표**: 평안함과 안식, 영감과 감동, 환영과 소속감
- **브랜드 컬러**:
  - 네이비 블루 `#1B3C8E` (메인 — 신뢰, 전통)
  - 그린 `#2E8B57` (포인트 — 생명, 성장)
  - 레드 `#D32F2F` (강조 — 성령, 열정)

### Aesthetic Direction
- **비주얼 톤**: 모던하고 세련된 — 깔끔한 레이아웃에 대담한 타이포그래피와 강한 대비
- **테마**: 라이트 모드 전용
- **레퍼런스**: 만나교회(manna.or.kr) — 시원시원한 레이아웃, 넉넉한 여백, 깔끔한 구성을 참고. 콘텐츠는 강남교회 자체 콘텐츠 사용
- **안티-레퍼런스**:
  - 화려하고 복잡한 사이트 (요소 과다, 시각적 혼란)
  - 올드한 교회 사이트 (2000년대 초반 느낌의 구식 디자인)
  - 지나치게 트렌디한 스타트업풍 (교회 정체성이 드러나지 않는 IT 기업 느낌)
- **타이포그래피**: 한글 가독성 우선, 세련되면서도 읽기 편한 서체. 고령 사용자를 위해 충분한 크기 유지
- **여백과 리듬**: 넉넉한 여백으로 시각적 여유와 평안함 연출

### Accessibility
- **기준**: WCAG 2.1 AA 이상
- **고령 사용자 배려**:
  - 본문 최소 16px 이상, 가급적 18px
  - 색상 대비비 4.5:1 이상 확보
  - 터치 타겟 최소 44px
  - 단순하고 예측 가능한 네비게이션
  - 모션 최소화 (prefers-reduced-motion 존중)

### Design Principles
1. **명료함 우선 (Clarity First)**: 정보는 한눈에 파악되어야 한다. 불필요한 장식보다 명확한 구조와 가독성을 우선한다.
2. **따뜻한 절제 (Warm Restraint)**: 모던하되 차갑지 않게. 세련된 미니멀리즘에 따뜻한 색감과 이미지로 교회의 온기를 담는다.
3. **누구나 쉽게 (Accessible by Default)**: 나이, 기기, 환경에 관계없이 모든 교인이 편하게 사용할 수 있어야 한다.
4. **신뢰의 일관성 (Consistent Trust)**: 컬러, 타이포, 간격, 톤이 전체 사이트에서 일관되게 유지되어 교회의 신뢰감을 시각적으로 전달한다.
5. **콘텐츠가 주인공 (Content is King)**: UI는 콘텐츠를 돋보이게 하는 그릇이다. 설교, 사진, 공지 등 실제 콘텐츠가 중심에 선다.
